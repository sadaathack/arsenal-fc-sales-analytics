# arsenal-fc-sales-analytics
## Arsenal FC Retail & Matchday Sales Analytics

This project analyzes Arsenal FC retail merchandise and matchday ticket sales in South Africa for 2024.

### Tools Used
- Microsoft Excel (Data cleaning & preparation)
- Power BI (Data modeling & visualization)

### Key Insights
- Identified top-performing products and categories
- Analyzed revenue trends across the year
- Compared matchday ticket sales vs merchandise
- Evaluated regional sales performance

### Skills Demonstrated
- Data cleaning
- Data modeling
- Power BI dashboards
- Business insight generation
